# My Learning Postの特徴
このアプリはユーザー登録時に複数のスキルを選択し、その中から学習したスキルを選択して学習時間・コメントを入力することで投稿が出来ます。<br>
投稿された学習時間を元に、マイページにて日別の学習時間をグラフ出力し日々の学習成果を可視化します。<br>
各スキルの総学習時間が一定時間を超えるとスキルにアイコンが付き、より学習の成果を実感できるようになります。<br>
